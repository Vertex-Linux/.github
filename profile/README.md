# Vertex Linux
Vertex linux is a linux distro (who could have thought) baised on Arch Linux and ArchISO, the first pre-release is out, it IS a pre-release though so it should not be used as your main operating system yet.

## Features:
- Custom package manager that lets you manage Pacman, AUR, and Flatpak packages in 1 tool! (Rewrite soon)
- Custom graphics card and package updater that lets you get Nvidia drivers and update our custom tools
- KDE Plasma desktop enviroment with custom theme (Soon to be replaced with our own custon one)
- Works great on older computers
