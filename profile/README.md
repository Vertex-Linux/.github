# Vertex Linux
Vertex linux is a linux distro (who could have thought) baised on Arch Linux and ArchISO, the first pre-release is out, it IS a pre-release though so it should not be used as your main operating system yet.

## Features:
- Custom package manager that lets you manage Pacman, AUR, and Flatpak packages in 1 tool!
- Custom graphics card and package updater that lets you get Nvidia drivers and update our custom tools
- Ultra lightweight Awesome baised desktop enviroment for ULTRA low ram usage
- Works great on older computers

## Graphics Support (Work In Progress list):
### AMD
- Vega: Untested
- RDNA 1: Untested
- RDNA 2+: Tested and wrorks fine
### Intel
- HD Graphics: Tested and works
- UHD Graphics: Untested but probably works
- Iris Graphics: Untested but probbaly works
### Nvidia
- All cards are untested, unsure if Nvidia even works.
